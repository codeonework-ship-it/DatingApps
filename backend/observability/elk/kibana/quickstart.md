# Kibana Quickstart

1. Open Kibana: `http://localhost:5601`
2. Create Data View: `dating-app-logs-*`
3. Time field: `@timestamp`
4. Discover query examples:
   - `service : "matching-svc"`
   - `user_id : "11111111-1111-1111-1111-111111111111" or userId : "11111111-1111-1111-1111-111111111111"`
5. Save a dashboard and use path `/app/dashboards` in control panel env.
